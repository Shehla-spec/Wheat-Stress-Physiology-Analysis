# Hormetic Effects of Ellagic Acid (EA) Priming on Wheat Resilience

## Executive Summary
This project investigates the efficacy of **Ellagic Acid (EA)** as a physiological rescue agent for winter wheat (*Triticum aestivum L.*, cv. Taskeen-22) under synergistic abiotic stress. While EA functions as a potent architectural modulator for root systems, the research identifies a critical **Source-Sink Gap** where vegetative gains fail to bypass reproductive stunting under combined drought (-0.1 MPa) and salinity (17.1 dS/m).
## Key Scientific Findings
**Biphasic Hormetic Response:** Identified **25 ppm EA** as the stimulatory threshold for root architecture ($p = 0.0018$), whereas 50 ppm induced an inhibitory response.
**Reproductive Floor:** Taskeen-22 maintains high inherent yield stability ($p = 0.50$) despite severe multi-stress conditions.
**Spike Stunting:** Identified a developmental bottleneck ($p = 0.018$) where EA-induced root gains (Source) do not translate to increased grain output (Sink).

## Technical Implementation (R)
The analysis utilizes **R (Tidyverse/ggplot2)** to process multivariate phenotypic data:
1. **Two-way ANOVA:** To isolate interaction effects between Stress Type and Priming Concentration.
2. **Interaction Modeling:** Visualizing how the 25 ppm threshold shifts across different stress environments.
3.**Data Vis:** Response heatmaps and phenotypic plasticity interaction plots.
